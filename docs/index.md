# Welcome to om_midi

欢迎来到 om_midi.jsx 的中文神奇世界。om_midi 事一个可以将 MIDI 文件转换为 AfterEffects 关键帧的脚本。常用于一键生成音骂抽动PV。

## 支持的AE版本

理论上支持 `CS5` 及以后的版本
